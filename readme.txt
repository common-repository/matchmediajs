== Description ==
MatchMedia.js is a javascript polyfill for testing media queries.
For more information visit  [GitHub](https://github.com/paulirish/matchMedia.js/)

From 1 October 2016 on, this plugin is no longer supported. Please use [Powerpack](http://www.ramoonus.nl/wordpress/powerpack) instead.

== Installation ==
1. Upload `matchmediajs/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
